# BanG Dream 角色 MBTI 标准化分析流程

## 目标

这份流程用于基于当前已经整理好的角色资料，对单个角色进行尽量稳定、可复核的 MBTI 分析。

分析时采用固定证据优先级：

1. `portrait.md` 作为核心依据
2. `card_stories.md` 作为重要验证
3. `story_slices.md` 作为补充验证

这样做的原因是：

- `portrait.md` 已经把外部设定、剧情功能、关系位置和本地资料分布做了初步归纳，最适合作为人物骨架。
- `card_stories.md` 更容易看到角色在私人场景、低压力场景、节日场景、回忆场景中的稳定表现，适合验证“这个人平时到底怎么想、怎么做、怎么表达”。
- `story_slices.md` 数量大、覆盖广，但更容易受到具体章节任务、剧情冲突、乐队分工和当下局势影响，因此更适合做补证，不适合直接主导类型判断。

## 证据权重

建议在实际判断时采用以下权重：

- 人物概述 `50%`
- 卡牌剧情 `30%`
- 剧情切片 `20%`

如果某个角色没有足够卡牌剧情，可以调整为：

- 人物概述 `60%`
- 剧情切片 `40%`

如果某个角色的外部设定很少，但卡牌剧情很多，可以调整为：

- 人物概述 `40%`
- 卡牌剧情 `40%`
- 剧情切片 `20%`

无论怎么调整，都不建议让剧情切片成为第一判断来源。

## 分析原则

### 1. 先看稳定人格，不先看单句台词

不能因为某个角色在一段剧情里愤怒、沉默、强势、崩溃，就直接把这种状态等同于 MBTI 结论。MBTI 要看的是相对稳定的注意力方向、信息处理方式、决策偏好和生活组织方式。

### 2. 先看人物功能，再看情绪表现

要优先关注：

- 这个角色在团队里通常承担什么功能
- 这个角色在关系中更常推动什么
- 这个角色在非冲突场景下更自然地怎么行动
- 这个角色面对不确定性、责任、关系压力时会回到什么习惯模式

情绪强度可以参考，但不能代替认知偏好。

### 3. 卡牌剧情优先验证“日常态”

卡牌剧情最适合判断：

- 角色私下是否更主动还是更保留
- 角色是否依赖个人感受、逻辑、责任或关系来做决定
- 角色是偏即兴、偏体验，还是偏计划、偏结构
- 角色在没有主线冲突压迫时，行为模式是否仍然稳定

### 4. 剧情切片优先验证“剧情位置”

剧情切片更适合判断：

- 角色在长期叙事里扮演什么位置
- 角色和谁绑定最深
- 角色在冲突里常采取什么手段
- 角色的推动方式是组织、调停、表达、观察、执行还是抽离

### 5. 允许保留备选类型

如果证据不足，不强行给唯一类型。可以采用：

- 主类型 + 次选类型
- 主类型 + 置信度
- 某条轴确定、某条轴待定

例如：

- `INFJ（中等置信度，备选 INFP）`
- `ENTP / ENFP 待定，优先看其决策方式是否更偏逻辑拆解还是价值驱动`

## 标准分析步骤

## 第一步：读取人物概述

先完整阅读 `portrait.md`，重点抽取以下四类信息：

- 外部设定基础
- 外部剧情基础
- 主线/乐团切片印证
- 卡牌剧情印证

读完后先写一段“人物核心摘要”，只回答四个问题：

1. 这个角色最核心的驱动力是什么
2. 这个角色在团队中的常态功能是什么
3. 这个角色在关系中的基本姿态是什么
4. 这个角色在压力下最容易出现什么偏移

这一段摘要是整份 MBTI 分析的起点，后面的判断都要回扣它。

## 第二步：先做四条维度的初判

基于 `portrait.md`，先对四条 MBTI 维度做一个初步判断。

### E / I 初判

看角色更习惯把能量投向哪里：

- 更偏 `E` 的信号：
  - 经常主动拉动关系和气氛
  - 倾向通过外部互动来确认方向
  - 在群体中思考，在群体中形成行动
  - 不回避公开表达和现场反馈

- 更偏 `I` 的信号：
  - 经常先内化再表达
  - 更依赖自己的内部节奏和内部判断
  - 在关系中常保留空间，不会持续向外扩张
  - 重要想法往往要经过沉淀才会外露

这里不要把“社恐”“安静”直接等于 `I`，也不要把“会说话”“舞台感强”直接等于 `E`。

### N / S 初判

看角色更习惯关注什么信息：

- 更偏 `N` 的信号：
  - 经常谈可能性、意义、理想、概念、方向感
  - 容易从局部联想到整体，从现实联想到象征
  - 对人、关系、舞台有抽象理解或世界观理解
  - 更容易围绕“为什么”“这意味着什么”组织行为

- 更偏 `S` 的信号：
  - 更关注现实执行、眼前经验、具体步骤、可操作细节
  - 处理问题时先抓住事实和当下条件
  - 比起愿景，更容易从经验和现成材料出发
  - 对节奏、动作、具体安排和实际效果更敏感

### T / F 初判

看角色决策时更依赖什么：

- 更偏 `T` 的信号：
  - 优先看结构、效率、合理性、规则、结果
  - 会主动切分问题并做逻辑判断
  - 关系重要，但不会天然把关系和谐放在第一位
  - 冲突里更容易先讲对错、方法、标准

- 更偏 `F` 的信号：
  - 优先看感受、价值、关系、意义、忠诚
  - 容易从“这样会伤害谁”“这样是否违背心意”出发
  - 决策往往带有明显的人际或内在价值取向
  - 冲突里更容易先看情感后果和关系状态

注意：嘴硬、冷脸、毒舌不等于 `T`；温柔、会照顾人也不自动等于 `F`。关键是决策基准，不是表面语气。

### J / P 初判

看角色面对生活和任务时更偏向哪种组织方式：

- 更偏 `J` 的信号：
  - 喜欢明确结构、计划、秩序、预期
  - 对“悬着”“没定下来”“不收束”的状态容忍度较低
  - 更愿意通过安排、推进、定规则来降低混乱
  - 责任感常体现为主动控制局面

- 更偏 `P` 的信号：
  - 更能接受临场变化和开放过程
  - 倾向保留空间，不急于封闭结论
  - 更容易顺着兴趣、现场反馈或当下灵感调整
  - 责任感不一定弱，但不一定通过“管理结构”来体现

## 第三步：用卡牌剧情做重要验证

在 `card_stories.md` 中重点看以下内容：

- `卡牌总览`
- 回忆/特训向条目
- 标准日常条目
- 高频同场角色

卡牌验证时，重点不是剧情大小，而是模式是否重复。

建议依次回答这些问题：

1. 角色在没有主线压力时，主动性是增强还是减弱
2. 角色在私人场景下，是更讲感受、讲逻辑、讲责任，还是讲体验
3. 角色是否更常主动组织事情，还是顺着情境推进
4. 角色在熟人面前和公开场合是否是同一种人
5. 角色反复稳定出现的行为习惯是什么

如果卡牌剧情反复支持 `portrait.md` 的初判，则该维度可以提高置信度。

如果卡牌剧情明显和 `portrait.md` 冲突，优先重新检查：

- 外部设定是否偏“剧情功能性描述”
- 当前初判是否把角色的社会面具误判成核心人格
- 角色是否存在公开人格与私下人格差异

## 第四步：用剧情切片做补充验证

在 `story_slices.md` 中重点看：

- `切片总览`
- 高频来源标题
- 高频同场角色
- 与关键关系人相关的切片
- 重要冲突、重要转折、重要表态的切片

剧情切片主要用来验证三件事：

1. 这个角色在长期叙事里如何推动剧情
2. 这个角色面对冲突时如何做选择
3. 这个角色的功能定位是否和 MBTI 判断一致

例如：

- 如果一个角色在大多数关键切片里都主动收束局面、建立规则、承担推进责任，那么 `J` 倾向通常会更强。
- 如果一个角色在关键切片里总是先抓住抽象意义、愿景、象征或身份命题，那么 `N` 倾向通常更强。
- 如果一个角色在冲突里总是优先保护关系、回应情绪或维系价值认同，那么 `F` 倾向通常更强。

但切片中的单次崩溃、单次强硬、单次退缩，不能单独拿来定型。

## 第五步：形成 MBTI 结论

完成三层证据核对后，按以下顺序落结论：

1. 先写“人物核心”
2. 再写四条维度判断
3. 再给完整 MBTI 类型
4. 最后给置信度和备选类型

建议采用如下格式：

### 人物核心

一句话概括角色最稳定的驱动力和功能位置。

### 四维判断

- `E / I`：结论 + 理由
- `N / S`：结论 + 理由
- `T / F`：结论 + 理由
- `J / P`：结论 + 理由

### 类型结论

- 主判断：`XXXX`
- 置信度：高 / 中 / 低
- 备选类型：如有则写

### 结论说明

用一段话解释为什么这个类型最贴合该角色，不要只列关键词。

## 第六步：记录证据出处

每次输出 MBTI 结论时，必须明确写出证据来自哪三层：

- `人物概述：` 引用了哪一部分总结
- `卡牌剧情：` 主要参考了哪类卡牌内容
- `剧情切片：` 主要参考了哪条关系线、哪类章节或哪类冲突

这样后续如果要复核、争议讨论或做批量校正，会方便很多。

## 批量分析时的固定顺序

为了保证所有角色口径一致，建议批量分析时一律按以下顺序处理：

1. 先读 `portrait.md`
2. 写人物核心摘要
3. 做四维初判
4. 读 `card_stories.md` 做重要验证
5. 读 `story_slices.md` 做补充验证
6. 形成主类型、备选类型、置信度
7. 写标准化输出

## 不建议的做法

以下做法建议避免：

- 看到某句名台词就直接判类型
- 用“外向=爱说话”“内向=不爱说话”这种表层规则直接套
- 用角色喜欢的舞台风格、服装风格直接推类型
- 把极端剧情状态当作常态人格
- 只看主线，不看卡牌
- 只看卡牌，不看人物概述
- 为了凑 16 型而无视证据不足

## 适合后续扩展的字段

如果这套流程确认可用，后续可以在每个角色的 MBTI 文档里追加这些字段：

- 功能堆栈倾向
- 压力状态下的失衡表现
- 在团体中的典型角色位
- 与高频关系人的类型互动方式
- 与备选类型的区分点

## 最终口径

一句话概括这套流程：

先用 `portrait.md` 确立人物骨架，再用 `card_stories.md` 判断稳定日常态，最后用 `story_slices.md` 验证剧情功能与冲突模式，据此形成带有证据出处的 MBTI 结论。
