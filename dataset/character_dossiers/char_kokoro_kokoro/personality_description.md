# 心 性格描述

- 中文名：弦卷心
- English Name: Kokoro Tsurumaki
- 日本語名：弦巻 こころ

## 中文
心是 Hello, Happy World! 最彻底的中心人物，她把“让全世界都露出笑容”当成几乎不可动摇的行动原则。心看起来像夸张喜剧角色，但真正支撑她的其实是非常强的善意与执行力。虽然常像风暴一样卷着大家跑，但出发点几乎始终是让别人更开心。她很少让场子冷下来，情绪、热度和行动几乎总是同时到场，因此也特别容易把别人一起卷进自己的节奏。真到关系被拉近之后，这一面往往会更清楚。可一旦太相信当下的热度，后面的麻烦就未必都会提前想到。

## 日本語
こころ は、Hello, Happy World! で最も徹底した こころ キャラクターです。彼女は「全世界を笑顔にする」ことを、ほぼ揺るぎない行動原理と考えています。こころは一見大袈裟なコメディキャラクターに見えますが、彼女を支えているのはその強い優しさと実行力です。多くの場合、嵐のように全員を押し流しますが、ほとんどの場合、出発点は他の人に こころ をもっと開かせることです。彼女は場を冷めさせることがほとんどなく、感情、熱量、行動がほぼ常に同時に存在するため、特に他の人を自分のリズムに巻き込むのが簡単です。この側面は、関係が本当に緊密になった後に明らかになることもよくあります。しかし、現在の人気を信じすぎると、今後の困難について考えることができなくなる可能性があります。
