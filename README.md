# BanG Dream! MBTI Match

基于 OJTS 题库制作的 BanG Dream! 角色性格拟合测试项目。

示例网站：[http://www.bangdream-mbti.cn/](http://www.bangdream-mbti.cn/)

## 声明

本项目代码由 AI 辅助生成，测试结果仅供图一乐和同人向娱乐体验使用，请不要将结果视为专业人格测评或官方角色设定结论。
